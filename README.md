🚀 AI Adaptive Learning Platform

An intelligent AI-powered adaptive learning system that personalizes education based on student performance, learning style, and real-time analytics.

📌 Features
🎯 Personalized Learning Paths
📊 Performance Dashboard
📈 Progress Tracking & Analytics
🤖 AI-Based Course Recommendation
🧠 Smart Quiz Generation
🔥 Weak Topic Detection
📚 Course Enrollment System
🛠️ Tech Stack
Frontend
React (TypeScript)
Tailwind CSS
Framer Motion
Recharts
Backend
FastAPI
SQLAlchemy
MySQL
AI Features
Performance-based recommendation
Adaptive quiz generation
Learning analytics
⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Anshika-123-tech/AI-adaptive-learning-platform.git
cd AI-adaptive-learning-platform
2️⃣ Backend Setup (FastAPI)
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Backend runs on:

http://127.0.0.1:8000


3️⃣ Frontend Setup (React)
cd client
npm install
npm run dev


Frontend runs on:

http://localhost:5173
📡 API Endpoints
http://127.0.0.1:8000/docs
POST /login → User login
GET /courses → Get all courses
POST /enroll → Enroll student
GET /progress/{student_id} → Progress summary
GET /performance/{student_id} → AI performance analysis
GET /generate-quiz/{student_id} → Generate quiz
POST /submit-quiz → Submit quiz
GET /smart-recommend/{student_id} → AI recommendation

🧠 How It Works
Student logs in
Enrolls in courses
Takes AI-generated quizzes
System tracks performance
Weak topics detected
AI recommends next learning path
